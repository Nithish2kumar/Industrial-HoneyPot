# Deception-Based Industrial Honeypot

A Python-based industrial honeypot for ICS networks that detects suspicious Modbus traffic and redirects attackers to a fake PLC.

## Features

- Real PLC Simulator
- Honeypot PLC
- Gateway Proxy
- Risk Scoring
- Attack Detection
- Traffic Redirection

## Technologies

- Python
- pymodbus
- Scapy
- Linux
